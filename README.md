# WorldInsights

A comprehensive web application for global data visualization, providing interactive insights into Population, GDP, Agriculture, and Health data using 3D visualization.

## Setup

### Environment

We use Conda for environment management.

```bash
conda create -n winsights python=3.11
conda activate winsights
pip install -r requirements.txt
```

### Running the App

```bash
python run.py
```
