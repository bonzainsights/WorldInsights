Data Usage Policy

Last Updated: December 25, 2025

1. Introduction

WorldInsights is a global data intelligence platform dedicated to providing reliable, verifiable, and reproducible insights from open world data. This Data Usage Policy explains how we source, process, store, and present data, as well as your rights and responsibilities when using our platform.

Our mission is to serve as a single trusted place for understanding global patterns, trends, and insights across economic, agricultural, social, health, and geographical perspectives.

2. Data Sources and Attribution

2.1 Primary Data Sources

WorldInsights aggregates data from reputable, publicly available sources:

  • World Bank Open Data: Economic indicators, development statistics, and global financial data
  • Food and Agriculture Organization (FAO): Agricultural production, food security, and nutrition data
  • World Health Organization (WHO): Global health statistics, disease surveillance, and healthcare metrics
  • NASA/NOAA: Climate data, environmental monitoring, and geospatial information

2.2 Data Licensing

All data sources used by WorldInsights are:

  • Publicly available under open data licenses
  • Properly attributed to their original sources
  • Subject to the terms and conditions of their respective providers
  • Accessed through official APIs or authorized channels

2.3 Attribution Requirements

When using WorldInsights data in publications, research, or reports, we encourage you to:

  • Cite the original data source (World Bank, FAO, WHO, NASA/NOAA)
  • Acknowledge WorldInsights as the platform used for analysis
  • Include the date of data access
  • Specify any transformations or calculations applied

3. Data Processing and Quality

3.1 API-First Approach

WorldInsights follows an API-first data strategy:

  • Data is fetched directly from source APIs in real-time or near-real-time
  • We do not store large datasets locally to avoid data staleness
  • This ensures you always access the most current data available
  • Caching is minimal and used only for performance optimization

3.2 Data Transformation

WorldInsights may apply the following transformations:

  • Normalization: Standardizing units and formats across sources
  • Aggregation: Combining data points for visualization and analysis
  • Interpolation: Filling gaps in time-series data (clearly marked)
  • Statistical Analysis: Calculating correlations, trends, and patterns

All transformations are documented and reproducible. The methodology is available to Researcher and Enterprise tier users.

3.3 Data Quality Assurance

We maintain data quality through:

  • Automated validation checks on incoming data
  • Anomaly detection for outliers and inconsistencies
  • Version tracking for data source updates
  • User reporting mechanisms for data issues

3.4 Reproducibility

WorldInsights is committed to scientific reproducibility:

  • All analytics are deterministic and reproducible
  • Data queries can be saved and re-executed
  • API endpoints provide consistent results for the same parameters
  • Methodology and assumptions are clearly documented

4. Data Storage and Retention

4.1 Database Technology

WorldInsights uses DuckDB for efficient data processing:

  • Optimized for analytical queries and large-scale data operations
  • Minimal local storage to maintain platform performance
  • Proper schema management for raw and processed data
  • Database schemas are versioned and documented

4.2 User-Generated Data

Data you create on WorldInsights (queries, visualizations, custom analyses) is:

  • Stored securely in our database
  • Associated with your user account
  • Accessible only to you (unless explicitly shared)
  • Retained according to our Privacy Policy
  • Deleted when you delete your account (after 30-day recovery period)

4.3 Data Immutability

Raw data from external sources is immutable:

  • Original data is never modified or deleted
  • Transformations create new derived datasets
  • Historical data remains accessible for longitudinal studies

5. Data Access and Permissions

5.1 Tier-Based Access

Data access varies by subscription tier:

Free Tier

  • Access to basic datasets (limited indicators)
  • Limited visualizations (5 per month)
  • Basic analytics capabilities
  • CSV export only
  • No API access

Researcher Tier ($29/month)

  • Access to all datasets and indicators
  • Unlimited visualizations
  • Advanced analytics and correlations
  • Machine learning features
  • Multiple export formats (CSV, JSON, Excel)
  • 3D globe visualization
  • Custom data queries
  • API access (10,000 calls/day)

Enterprise Tier (Custom)

  • All Researcher features
  • Custom data integrations
  • Unlimited API calls
  • White-label options
  • On-premise deployment
  • Custom data pipelines

5.2 Role-Based Access Control

User roles determine platform capabilities:

  • User: Standard access based on subscription tier
  • Researcher: Enhanced access to advanced features (independent of subscription)
  • Admin: Platform administration, user management, and system configuration

Note: User roles are managed by administrators and are separate from subscription tiers.

6. API Usage Policy

6.1 API Access

API access is available to Researcher and Enterprise tier subscribers:

  • RESTful API with JSON responses
  • Authentication via API keys
  • Comprehensive documentation available
  • Rate limiting based on subscription tier

6.2 Rate Limits

  • Free Tier: 100 API calls per day
  • Researcher Tier: 10,000 API calls per day
  • Enterprise Tier: Unlimited API calls

Exceeding rate limits will result in temporary throttling. Contact us for higher limits.

6.3 Acceptable API Use

When using our API, you must:

  • Respect rate limits and not attempt to circumvent them
  • Not resell or redistribute API data without authorization
  • Implement proper error handling and retry logic
  • Cache responses appropriately to minimize redundant calls
  • Attribute data sources in applications using our API

7. Data Export and Portability

7.1 Export Formats

WorldInsights supports multiple export formats:

  • CSV: Available to all tiers
  • JSON: Researcher and Enterprise tiers
  • Excel (XLSX): Researcher and Enterprise tiers
  • API: Researcher and Enterprise tiers

7.2 Data Portability Rights

You have the right to:

  • Export your user-generated content at any time
  • Download your visualizations and analyses
  • Retrieve your query history
  • Request a complete data export upon account deletion

8. Data Privacy and Security

8.1 User Data Protection

Your personal data and usage patterns are protected:

  • Encrypted transmission (SSL/TLS)
  • Secure storage with access controls
  • Regular security audits and vulnerability testing
  • Compliance with data protection regulations (GDPR, etc.)

8.2 Data Anonymization

When we use aggregated usage data for platform improvement:

  • Personal identifiers are removed
  • Data is aggregated across multiple users
  • Individual users cannot be identified

8.3 Third-Party Data Sharing

We do not share your data with third parties except:

  • As required by law
  • With your explicit consent
  • With service providers under strict confidentiality agreements

See our Privacy Policy for complete details.

9. Data Accuracy and Limitations

9.1 Data Accuracy

While we strive for accuracy:

  • Data accuracy depends on source providers
  • We cannot guarantee 100% accuracy or completeness
  • Data may be revised or updated by source providers
  • Users should verify critical data independently

9.2 Known Limitations

  • Data availability varies by country and indicator
  • Historical data may have gaps or inconsistencies
  • Real-time data may have delays based on source update frequency
  • Some indicators are estimates or projections

9.3 Disclaimer

WorldInsights provides data "as is" without warranty. We are not responsible for decisions made based on our data. Always verify data for critical applications.

10. Intellectual Property

10.1 Source Data

Original data belongs to source providers (World Bank, FAO, WHO, NASA/NOAA) and is subject to their respective licenses.

10.2 Platform Analytics

WorldInsights owns the intellectual property rights to:

  • Our analytical algorithms and methodologies
  • Platform software and visualizations
  • Derived insights and correlations
  • User interface and design

10.3 User Content

You retain ownership of your analyses and visualizations created on WorldInsights. By using our platform, you grant us a license to host and display your content as necessary to provide our services.

11. Updates to This Policy

We may update this Data Usage Policy to reflect:

  • Changes in data sources or methodologies
  • New features or capabilities
  • Legal or regulatory requirements
  • User feedback and best practices

Material changes will be communicated via email and posted on this page with an updated "Last Updated" date.

12. Contact Us

For questions about our data practices, sources, or this policy:

  • Email: data@worldinsights.bonzainsights.com
  • Contact Form: Contact Us
  • Documentation: Platform Documentation

WorldInsights is committed to transparency, reproducibility, and scientific credibility in all our data practices.
